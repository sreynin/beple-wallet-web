#### SCN-HOM-001
| 항목 | 내용 |
|------|------|
| Scenario ID | SCN-HOM-001 |
| 우선순위 | P0 |
| 대상 기능 | F-HOM-001 |
| 대상 화면 | HOM-001 |
| 제목 | Beple Money 잔액 정상 표시 |

**사전 조건**
- 사용자가 온보딩을 완료한 상태
- Beple Money 잔액이 1원 이상인 계정으로 로그인

**Happy Path**
1. 앱 실행 후 홈 화면(HOM-001) 진입
2. 상단 잔액 카드에 "Beple Money" 레이블과 KRW 잔액이 표시됨

**예상 결과**
- 잔액이 정확한 금액(원화, 천 단위 콤마 포함)으로 표시된다
- Primary/100 배경 카드 스타일이 적용된다

**예외 케이스**
- 잔액이 0원인 경우 → HOM-002(Empty 화면)로 분기되어야 함

**예외 케이스**
- 네트워크 오류로 잔액 조회 실패 시 → 오류 상태 UI 또는 재시도 안내 표시
