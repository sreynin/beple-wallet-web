## P0 — 핵심 기능

---

### SCN-WDR-001
| 항목 | 내용 |
|------|------|
| 시나리오명 | 최초 이용 — 약관 동의 후 QR 스캔까지 전체 플로우 진행 |
| 연결 기능 | F-WDR-001 |
| 연결 화면 | WDR-000 → WDR-005 → WDR-001 |
| 우선순위 | P0 |

**Happy Path**
1. ATM 출금 이용 이력이 없는 상태에서 ATM 출금 기능 진입
2. WDR-000 COATM 이용약관 화면이 표시된다
3. 전체 동의 체크박스 체크 — 개별 약관(COATM 이용약관 필수, 출금 수수료 안내 필수) 모두 자동 체크됨을 확인
4. "동의하고 계속하기" 버튼 활성화 확인 후 탭
5. WDR-005 ATM 출금 안내 화면으로 이동 확인
6. "QR 스캔하기" 버튼 탭
7. WDR-001 ATM QR 스캔 화면으로 이동 확인

**기대 결과**
- 약관 미동의 시 "동의하고 계속하기" 버튼이 비활성화 상태임
- WDR-000 → WDR-005 → WDR-001 순서로 화면 전환됨

---

### SCN-WDR-002
| 항목 | 내용 |
|------|------|
| 시나리오명 | 재이용 — WDR-005 안내 화면 직접 진입 |
| 연결 기능 | F-WDR-001 |
| 연결 화면 | WDR-005 → WDR-001 |
| 우선순위 | P0 |

**Happy Path**
1. ATM 출금 이용 이력이 있는 상태에서 ATM 출금 기능 진입
2. WDR-000 약관 화면이 표시되지 않고 WDR-005 ATM 출금 안내 화면이 바로 표시됨
3. "QR 스캔하기" 버튼 탭 → WDR-001로 이동 확인

**기대 결과**
- 이용 이력 있는 경우 WDR-000을 건너뛰고 WDR-005로 직접 진입함

### SCN-WDR-003
| 항목 | 내용 |
|------|------|
| 시나리오명 | ATM QR 스캔 성공 후 금액 입력 화면 전환 |
| 연결 기능 | F-WDR-001, F-WDR-002 |
| 연결 화면 | WDR-001 → WDR-002 |
| 우선순위 | P0 |

**Happy Path**
1. WDR-001 화면에서 카메라 뷰가 정상 표시됨을 확인
2. ATM 화면의 유효한 QR 코드를 스캔
3. QR 스캔 성공 후 WDR-002 출금 금액 입력 화면으로 자동 이동

**기대 결과**
- QR 스캔 영역(4모서리 가이드) 정상 표시
- 스캔 성공 즉시 WDR-002로 이동함

### SCN-WDR-004
| 항목 | 내용 |
|------|------|
| 시나리오명 | 출금 금액 입력 — 유효 금액 입력 후 확인 화면 이동 |
| 연결 기능 | F-WDR-002 |
| 연결 화면 | WDR-002 → WDR-003 |
| 우선순위 | P0 |

**Happy Path**
1. WDR-002 화면에서 현재 잔액이 표시됨을 확인
2. 숫자 키패드로 "100000" (10만원) 입력
3. "다음" 버튼 활성화 확인 후 탭
4. WDR-003 출금 확인 화면으로 이동 확인

**기대 결과**
- 금액이 대형 폰트로 중앙 표시됨
- 1만원 이상, 50만원 이하, 1만원 단위인 경우 "다음" 버튼 활성화

### SCN-WDR-005
| 항목 | 내용 |
|------|------|
| 시나리오명 | PIN 인증 성공 후 출금 완료 |
| 연결 기능 | F-WDR-003, F-WDR-004 |
| 연결 화면 | WDR-003 → WDR-004 |
| 우선순위 | P0 |

**Happy Path**
1. WDR-003 화면에서 출금 금액, ATM 정보, 수수료(있는 경우) 확인
2. 숫자 키패드로 PIN 6자리 입력
3. PIN 도트 인디케이터가 6자리 모두 채워짐 확인
4. PIN 인증 성공 후 WDR-004 출금 완료 화면으로 이동

**기대 결과**
- WDR-003 확인 카드에 출금 금액, ATM 정보가 정확히 표시됨
- PIN 입력 시 도트 인디케이터가 순서대로 채워짐
- 인증 성공 후 WDR-004로 이동하며 완료 아이콘, 출금 금액, 잔액 표시됨

**기대 결과** — 필수 약관 중 하나라도 미동의 시 버튼 비활성화
