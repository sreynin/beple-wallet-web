### SCN-SET-001 — 설정 메뉴 진입 및 항목 표시

| 항목 | 내용 |
|------|------|
| Scenario ID | SCN-SET-001 |
| 기능 | F-SET-001 설정 메뉴 |
| 화면 | SET-001 set-menu.html |
| 우선순위 | P0 |
| 유형 | Happy Path |

**전제 조건**
- 사용자가 로그인된 상태

**테스트 단계**

| 순서 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | 설정 화면 진입 | TopBar에 "설정" 타이틀 표시 |
| 2 | 사용자 정보 카드 확인 | 이름 및 이메일 또는 전화번호 표시 |
| 3 | 메뉴 리스트 확인 | 언어 설정 / 알림 설정(토글) / 생체인증(토글) / PIN 변경 / 로그아웃 5개 항목 전부 표시 |
| 4 | 각 항목 UI 확인 | 아이콘 + 라벨 + 우측 ">" 또는 토글이 항목별로 올바르게 표시 |

**합격 기준**
- 5개 메뉴 항목이 모두 표시된다.
- 알림 설정, 생체인증 항목에 토글이, 나머지에 ">"가 표시된다.
- 사용자 정보 카드에 실제 데이터가 표시된다.

**예외 케이스**
- 사용자 정보 카드에 이름이 없는 경우 이메일 또는 전화번호만 표시된다.
- 메뉴 항목 중 하나라도 누락되면 검수 실패로 기록한다.

### SCN-SET-002 — 언어 변경 전체 플로우

| 항목 | 내용 |
|------|------|
| Scenario ID | SCN-SET-002 |
| 기능 | F-SET-002 언어 변경 |
| 화면 | SET-001 → SET-002 |
| 우선순위 | P0 |
| 유형 | Happy Path |

**전제 조건**
- 설정 메뉴 화면 표시 상태
- 현재 언어: 한국어

**테스트 단계**

| 순서 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | "언어 설정" 항목 탭 | SET-002 언어 변경 화면으로 이동 |
| 2 | 화면 구성 확인 | TopBar 뒤로가기 + "언어 설정" 타이틀, 5개 언어 리스트, 현재 언어 체크 표시, "변경하기" 버튼(Primary, XLarge) |
| 3 | English 항목 탭 | English에 체크 표시 이동 |
| 4 | "변경하기" 버튼 탭 | 앱 전체 UI가 English로 즉시 변경, SET-001로 복귀 |
| 5 | SET-001 메뉴 라벨 확인 | "Settings" 등 English로 표시 |
| 6 | 앱 재시작 후 확인 | 변경된 언어(English) 유지 |

**합격 기준**
- 5개 언어(한국어, English, 中文, 日本語, Español)가 모두 리스트에 표시된다.
- "변경하기" 탭 즉시 앱 전체 UI 언어가 변경된다.
- 앱 재시작 후에도 변경된 언어가 유지된다.

**예외 케이스**
- 현재 사용 중인 언어를 그대로 선택하고 "변경하기"를 탭해도 오류 없이 정상 동작한다.
- 언어를 선택하지 않고 뒤로가기를 탭하면 언어가 변경되지 않고 SET-001로 돌아간다.
- 5개 언어 전체가 리스트에 표시되지 않으면 검수 실패로 기록한다.

### SCN-SET-003 — PIN 변경 전체 플로우

| 항목 | 내용 |
|------|------|
| Scenario ID | SCN-SET-003 |
| 기능 | F-SET-005 PIN 변경 |
| 화면 | SET-001 → SET-003 |
| 우선순위 | P0 |
| 유형 | Happy Path |

**전제 조건**
- 설정 메뉴 화면 표시 상태
- 현재 PIN: 123456

**테스트 단계**

| 순서 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | "PIN 변경" 항목 탭 | SET-003 PIN 변경 화면으로 이동 |
| 2 | 화면 구성 확인 | TopBar(뒤로가기) + "PIN 변경" 타이틀, 3단계 스텝 인디케이터(1단계 활성), Page Title, 6자리 PIN 도트 인디케이터, 숫자 키패드 |
| 3 | 현재 PIN 6자리 입력 (123456) | 도트 6개 채워짐, 자동으로 2단계 전환, 스텝 인디케이터 2단계 활성 |
| 4 | 새 PIN 6자리 입력 (789012) | 도트 6개 채워짐, 자동으로 3단계 전환, 스텝 인디케이터 3단계 활성 |
| 5 | 새 PIN 확인 6자리 입력 (789012) | PIN 변경 완료, SET-001로 복귀 또는 성공 안내 표시 |

**합격 기준**
- 3단계 플로우(현재 PIN → 새 PIN → 새 PIN 확인)가 순서대로 진행된다.
- 완료 후 설정 화면으로 복귀한다.
- 변경된 새 PIN으로 이후 인증이 가능하다.

**예외 케이스**
- Step 1에서 현재 PIN을 잘못 입력하면 오류 메시지가 표시되고 도트 인디케이터가 초기화된다.
- Step 3에서 Step 2와 다른 PIN을 입력하면 오류 메시지가 표시되고 Step 3가 초기화된다.
- 새 PIN이 현재 PIN과 동일한 경우 정책에 따른 오류 처리 여부를 확인한다.

### SCN-SET-004 — 로그아웃 전체 플로우

| 항목 | 내용 |
|------|------|
| Scenario ID | SCN-SET-004 |
| 기능 | F-SET-006 로그아웃 |
| 화면 | SET-001 (Modal 포함) → ONB-001 |
| 우선순위 | P0 |
| 유형 | Happy Path |

**전제 조건**
- 설정 메뉴 화면에서 로그인 상태

**테스트 단계**

| 순서 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | "로그아웃" 항목 탭 | 로그아웃 확인 Modal 표시 |
| 2 | Modal 구성 확인 | 반투명 배경(dimmed) + 중앙 카드, 타이틀 "로그아웃", 본문 "정말 로그아웃 하시겠습니까?", 좌측 "취소"(Secondary), 우측 "로그아웃"(Danger) |
| 3 | "로그아웃" 버튼 탭 | 세션 종료 후 ONB-001(언어 선택) 화면으로 이동 |
| 4 | 로그아웃 후 뒤로가기 탭 | 이전 인증 화면으로 이동하지 않음 |

**합격 기준**
- 로그아웃 탭 시 확인 Modal이 반드시 표시된다.
- "로그아웃" 버튼 탭 시 세션이 완전히 종료된다.
- ONB-001(언어 선택) 화면으로 이동한다.
- 로그아웃 후 세션이 복구되지 않는다.

**예외 케이스**
- "취소" 버튼을 탭하면 Modal이 닫히고 SET-001 화면이 유지된다.
- Modal 외부(dimmed) 영역을 탭했을 때 Modal 닫힘 또는 유지를 확인한다. 로그아웃은 실행되지 않아야 한다.
- 로그아웃 완료 후 딥링크로 내부 화면 접근 시도 시 로그인 화면으로 리다이렉트된다.
