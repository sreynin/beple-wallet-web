- **우선순위:** P0
- **연결 기능:** F-PAY-001
- **연결 화면:** PAY-001
- **제목:** QR 스캔 — 정상 진입 및 스캔 성공

**전제 조건**
- 앱 로그인 상태
- 카메라 권한 허용

**Happy Path**

| 단계 | 행동 | 기대 결과 |
|------|------|-----------|
| 1 | 홈에서 QR 결제 진입 | PAY-001(QR 스캔) 화면 표시 |
| 2 | 화면 상단 TopBar 확인 | 타이틀 "QR 결제", X(닫기) 버튼 노출 |
| 3 | 카메라 뷰 확인 | 전체 화면 카메라 활성화, 4모서리 가이드 프레임 표시 |
| 4 | 안내 텍스트 확인 | "가맹점 QR 코드를 스캔해주세요" 흰색 텍스트 노출 |
| 5 | 가맹점 QR 코드 인식 | PAY-002(결제 확인) 화면으로 자동 전환 |

**예외 케이스**

| 케이스 | 행동 | 기대 결과 |

| E-2 | 유효하지 않은(비가맹점) QR 코드 스캔 | 오류 안내 메시지 표시, 재스캔 가능 상태 유지 |
| E-3 | X 닫기 버튼 탭 | 이전 화면(홈)으로 복귀 |
